# Security

## Forms

1. Use React Hook Form and Zod for forms.

## Server Actions

1. Ensure that the user is logged in with Clerk: [Protecting Next.js Server Actions](https://clerk.com/docs/references/nextjs/server-actions)
2. Perform authorization checks if the user is OK to be perfoming the action.
3. Ensure that the data is valid by performing Zod validation.
